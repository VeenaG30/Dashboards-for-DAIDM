📦 Supply Chain Resilience & Inventory Analytics Dashboard
🚀 Project Overview
This project provides an end-to-end analytical solution for managing supply chain disruptions. By leveraging historical data, the dashboard provides actionable insights into inventory mismanagement, demand forecasting errors, and the financial impact of external disruptions (weather, tariffs, and logistics delays).

The application is built using Python and Streamlit, featuring a dynamic interface that allows users to toggle between light and dark modes and explore data through four analytical lenses: Diagnostic, Forecasting, Predictive, and Prescriptive.

📊 Key Features
Diagnostic Analytics: Root cause analysis of lost sales and customer satisfaction drops linked to lead times and weather events.
Demand Forecasting: Visualization of forecast accuracy vs. actual demand trends.
Predictive Analytics: Identification of high-risk SKUs prone to stock-outs based on current inventory levels and reorder points.
Prescriptive Analytics: Data-driven recommendations to optimize safety stock and reduce excess inventory holding costs.
Dynamic UI: Interactive filters and a theme-toggle feature for improved user experience.
🛠 Tech Stack
Language: Python
Framework: Streamlit
Data Manipulation: Pandas, NumPy
Visualization: Plotly
Deployment: Streamlit Community Cloud
